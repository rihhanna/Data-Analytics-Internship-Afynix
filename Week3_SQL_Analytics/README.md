# 📊 Week 3: SQL for Analytics

## Project: Superstore SQL Analysis

### Objective
Analyze retail sales data using SQL queries to answer business questions and generate actionable insights.

## Dataset
- Sample Superstore Dataset
- Records: 9,994
- Features: 21

## Tools Used
- Python
- SQLite
- Pandas
- Jupyter Notebook

## SQL Skills Practiced
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (SUM, AVG, COUNT)
- LIMIT
- Subqueries

## Business Questions Answered
- What are the total sales and profit?
- Which categories generate the most revenue?
- Which regions are the most profitable?
- Who are the top customers?
- Which products generate the highest sales?
- Which transactions resulted in losses?
- Which products sell above the average sales value?

## Key Insights
- Technology contributes the highest sales.
- Some orders generate losses despite high sales.
- Customer purchasing behavior varies significantly.
- Regional profitability differs, highlighting opportunities for targeted business strategies.

## Outcome
This project demonstrates practical SQL skills for business analytics by extracting meaningful insights from a retail sales dataset using SQLite.
