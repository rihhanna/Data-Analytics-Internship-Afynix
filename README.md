# 📊 Data Analytics Internship – Afynix Digital

This repository contains my work and projects completed during the **6-week Data Analytics Internship at Afynix Digital**. The internship focuses on developing practical skills in data cleaning, visualization, SQL analytics, dashboarding, and business analysis.

## 🚀 Internship Roadmap

### 📌 Week 1 – Data Cleaning & Preparation

**Project:** Retail Sales Dataset Cleaning

* Data inspection and understanding
* Handling missing values and duplicates
* Outlier detection
* Data type conversion
* Exploratory Data Analysis (EDA)

### 📌 Week 2 – Data Visualization

**Project:** Superstore Sales Visualization

* Trend analysis
* Category comparison
* Regional performance analysis
* Profit vs Sales insights
* Data storytelling with visualizations

### 📌 Week 3 – SQL for Analytics

**Project:** E-Commerce Database Queries

* Aggregations and GROUP BY
* Table joins
* Subqueries
* Customer and product analysis
* Query optimization

### 📌 Week 4 – Dashboarding

**Project:** Power BI / Tableau Sales Dashboard

* KPI development
* Interactive dashboards
* Filters and slicers
* Drill-down analysis
* Business insights

### 📌 Week 5 – Capstone Analytics Project

**Project:** Marketing Campaign Analysis

* Data cleaning and exploration
* Customer segmentation
* Campaign performance analysis
* Visualization and reporting
* Actionable recommendations

### 📌 Week 6 – Project Completion & Submission

* Final project review
* Repository organization
* LinkedIn project posts
* Internship submission
