# Week 1: Data Cleaning & Preparation

## Project: Retail Sales Dataset Cleaning

### Objective
Prepare a retail sales dataset for analysis by handling data quality issues and performing exploratory data analysis.

## Tasks Performed

- Loaded and inspected the dataset
- Checked data types and summary statistics
- Converted date columns to datetime format
- Checked for missing values
- Checked for duplicate records
- Performed exploratory data analysis
- Analyzed sales distribution
- Identified top-selling products
- Compared sales across categories and regions

## Key Findings

- Dataset contains 9,994 records and 21 columns.
- No missing values were found.
- No duplicate records were found.
- Sales range from 0.44 to 22,638.48.
- Profit ranges from -6,599.98 to 8,399.98.
- Discounts reach up to 80%.
